# my-music
#### 描述
使用ES6+webpack+react+redux+react-router搭建自己的酷狗音乐。

该项目的目的是为了学习应用redux。

#### 技术栈
ES6 + webpack + react + redux + react-router

#### API来源
https://code.it919.cn/2017/07/kugou-api/

#### 使用
```
//克隆
git clone https://github.com/tian-cai/my-music.git

//安装依赖
npm install

//运行
npm run start

//打包
npm run build

```
